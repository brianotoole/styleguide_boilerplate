# [SMG Web: Brand Styleguide Template]

Overview

This repo servers as the boilerplate for web projects involving an online brand styleguide / identity.(http://startbootstrap.com/).

Getting Started

To use this template, do the following: * Download the latest repo. * Install it into project directory. * Uses Sass; Codekit or similar compiler is recommended * Sass compiles to: css/main.css (scss/main.scss = main stylesheet) * Add / Remove or Edit content with your site’s information * That’s all folks.
